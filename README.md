# meli-project
Client/server app para búsqueda y visualización de productos

**Stack:**
  - Angular.
  - Express.
  - NodeJS

**Installation**
  - `git clone https://github.com/JoaquinR1/meli-project.git` 

**Iniciar server:**
  - `cd server`
  - `npm i`
  - `npm start`

**Iniciar client:**
  - `cd client`
  - `npm i`
  - `npm start`
